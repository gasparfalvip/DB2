**create**: Táblák létrehozása és feltöltése. (hogy legyen min kipróbálnom az eljárásokat és funkciókat) <br />
**insert**: Tárolt eljárások adatok felvitelére, trigger és hozzátartozó sequence az automatikus kulcs generáláshoz. (+funkció következő id lekérdezésére) <br />
**updates**: Tárolt eljárások adatok módosítására, trigger egy módosítás kontrollálására. <br />
**delete**: Tárolt eljárások adatok törlésére. <br />
**nalózás**: Triggerek módsítási események naplózására, illetve napló táblák létrehozása. <br />
**lekerdezes_funkciok**: Fügvények adott rekord adatainak lekérdezésére id szerint. Függvények aggregált értékek lekérdezésére (adott nevű TKhoz tartozó: filmek száma, illetve legdrágább film). <br />
**packegeles**: Táblákhoz tartozó funkcionalitások csomagolása. <br />
